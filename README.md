# 🦈 xShark Sistemas: Soluções Tecnológicas Inteligentes

> **Status do Projeto:** Concluído ✅  
> **Deploy:** [Vercel](https://xshark-sistemas.vercel.app/)

A **xShark Sistemas** é uma landing page institucional desenvolvida para uma empresa de Sistema EPR.
O projeto foca em uma estética moderna e agressiva, utilizando uma paleta de cores claras com acentos contrastantes em azul para transmitir autoridade,
inovação e segurança no setor de TI.

## 🚀 Funcionalidades e Diferenciais

* **Design de Alto Impacto:** Interface desenvolvida com foco em UI Design moderno, utilizando elementos visuais que remetem à agilidade e precisão.
* **Navegação Responsiva:** Site totalmente otimizado para dispositivos móveis, tablets e desktops (Mobile-First).
* **Seções Estratégicas:** Apresentação de serviços, diferenciais competitivos e chamadas para ação (CTAs) estrategicamente posicionadas para geração de leads.
* **Arquitetura CSS Modular:** Estilização organizada por seções, facilitando a manutenção e garantindo performance de carregamento.
* **SEO & Performance:** Estrutura semântica para melhor ranqueamento em motores de busca e otimização de ativos para carregamento rápido via Vercel.

## 🧠 Aprendizados Técnicos

Neste desenvolvimento, apliquei conceitos avançados de:
1. **Composição Visual:** Uso de sombras, gradientes e tipografia para criar uma hierarquia visual clara e profissional.
2. **UX Writing & Layout:** Organização de conteúdo de forma que a leitura seja fluida e o objetivo de negócio seja alcançado.
3. **Gestão de Deploy:** Ciclo completo de publicação e configuração de ambiente de produção.
